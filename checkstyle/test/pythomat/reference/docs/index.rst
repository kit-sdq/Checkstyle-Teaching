Welcome to pythomat documentation
=================================

Pythomat is a python module for writing better test scripts for the Praktomat
system faster and more easily.

The pythomat package consists of two primary modules: fakeomat and pythomat.
The module fakeomat can be used to simulate the Praktomat system on a local
machine. The module pythomat is a module providing support for writing various
kinds of tests for this system.
