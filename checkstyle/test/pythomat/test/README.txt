========================================
Pythomat - The Praktomat Support Package
========================================

The package contains two modules, fakeomat and pythomat:

- The fakeomat module contains code for simulating a Praktomat run. File
  copying, compiling and execution of script checkers are supported by it.

- The pythomat module contains code that can be used to write script checkers
  more easily and more consitently.


Pythomat provides various helper class and functions for writing checkers for
the Praktomat web service. Using these classes ensures the visual consistency
and a safe execution of the checkers.
